Brand fetch chain — runs before every HTML output
1. Fetch bsn-tools:bsn-brand via MCP (primary)
2. If MCP unavailable → read bsn-brand-fallback.md (local fallback)
3. If both unavailable → apply minimum brand tokens and flag in HTML version header comment
